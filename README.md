# FactuStart Version 1

Sistema de facturación web desarrollado como proyecto académico.

## Tecnologías utilizadas

- Java JSP
- Jakarta Servlets
- MySQL 8
- NetBeans 20
- GlassFish 7
- HTML
- CSS
- JavaScript

## Funcionalidades

- Registro e inicio de sesión de usuarios
- Registro de datos del negocio
- Emisión de boletas y facturas
- Cálculo automático de subtotal, IGV y total
- Historial de comprobantes
- Reportes de ventas
- Configuración de usuario
- Cierre de sesión

## Instalación

1. Descargar el proyecto desde GitHub.
2. Importar el archivo `FactuStartBaseDatos.sql` en MySQL Workbench.
3. Abrir el proyecto en NetBeans.
4. Configurar la conexión a la base de datos.
5. Ejecutar el proyecto en GlassFish.
